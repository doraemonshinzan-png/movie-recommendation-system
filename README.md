# 🎬 Movie Recommendation System

An AI-based movie recommender built using Python and Machine Learning.

## 🚀 Features

* Recommend similar movies
* Fast and simple algorithm
* CLI + Streamlit UI

## 🛠 Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

## ▶️ Run Locally

pip install -r requirements.txt
python main.py

## 🌐 Run UI

streamlit run app.py

## 📌 Project Type

Machine Learning / AI Project
